## ğŸ” Food Delivery Web Application (MERN Stack)

A full-stack **Food Delivery Web Application** built with the **MERN stack**
(MongoDB, Express.js, React.js, Node.js).

The application allows users to browse food items, add them to the cart, and
place orders seamlessly. It includes secure **user authentication**, **cart
management**, and **Stripe payment integration**, along with an **admin
dashboard** for managing food items and orders.

---

## âœ¨ Features

- User authentication (Sign up & Login)
- Browse food items with images and details
- Add, update, and remove items from cart
- Real-time cart total calculation
- Place food orders securely
- Stripe payment gateway integration
- View order history
- Admin dashboard for managing food items
- Admin order management
